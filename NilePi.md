## 📖 إيه هو NilePi؟

**NilePi** مساعد ذكاء اصطناعي كامل بيشتغل **محليًا بالكامل** على جهاز لابتوب عادي (RTX 4050 بـ 6GB VRAM بس!) — من غير ما بايت واحد من بياناتك يخرج من الجهاز. بيفهم العربي والإنجليزي، بيقرا مستنداتك ويجاوبك منها **بالمصدر ورقم الصفحة**, بيفهم الصور ويقرا اللي فيها، بيكتب أكواد كاملة جاهزة للتشغيل، بيولّد صور، بيعمل تقارير Word أكاديمية، وبيفتكرك حتى بعد ما تقفله — وكمان **موقع كامل متعدد المستخدمين** بحسابات وشات ودعم فني ولوحة أدمن.

### 💬 جرب تقوله:

> 📄 *"لخصلي ملف المحاضرة اللي رفعته"* → ملخص **بالمصدر ورقم الصفحة**
> 📊 *"متوسط السعر لكل حي في ملف الـ CSV؟"* → حساب فعلي بـ pandas مش تخمين
> 💻 *"اكتبلي سكريبت بايثون ينظم الصور في فولدرات حسب التاريخ"* → كود كامل جاهز للتشغيل
> 🎨 *"ارسملي غروب على النيل بألوان دافية"* → صورة حقيقية في **0.9 ثانية**
> 🖼️ *(ترفع سكرين شوت)* *"مكتوب فيها إيه؟"* → قراءة النص عربي/إنجليزي
> 🧠 *"افتكر إن مشروع تخرجي عن الطاقة الشمسية"* → هيفتكرها حتى بعد ما تقفل الجهاز

## ✨ الفيتشرز الكاملة

### 📄 اسأل في مستنداتك — بمصدر دقيق
- بيبلع **PDF (حتى الممسوح ضوئيًا بالـ OCR) • Word • PowerPoint • Excel • CSV**
- كل إجابة بتيجي **بالمصدر بالظبط**: `(المصدر: TEST.pdf، الصفحة: 3)` / `(الشريحة: 5)` / `(الشيت: Sheet1)`
- لخص، اسأل، قارن بين ملفات، اطلب أسئلة امتحان (MCQ أو سؤال وجواب) من أي ملف
- أسئلة الـ metadata فورية: "اسم الملف اللي رفعته إيه؟" / "كام صفحة؟" — إجابة لحظية من غير موديل

### 📊 تحليل بيانات حقيقي — مش تخمين
- "متوسط السعر لكل حي في الـ CSV؟" → **حساب فعلي بـ pandas** مش هلوسة نصية
- متوسطات، مجاميع، أعلى/أقل قيمة، تجميع حسب فئة (group by) — كله محسوب من جداولك

### 💻 مبرمج جاهز تحت أمرك
- بيكتب **كود كامل قابل للتشغيل** بموديل مخصوص للبرمجة (`qwen2.5-coder`)
- **فحص جودة تلقائي**: أي ملف Python بيتفحص بالـ AST — لو طلع ناقص (من غير entry point) بيتعاد تلقائيًا
- فحص syntax قبل الحفظ — مستحيل يتحفظ ملف بايثون مكسور
- الكود بيظهر **جوة الشات** منسق وجاهز للنسخ، مش مجرد "اتحفظ في المسار الفلاني"
- بيفهم الأكواد كمان: ابعتله أي كود واسأله "اشرحلي ده" أو "فين الغلط؟"

### 🎨 توليد صور محلي — في أقل من ثانية
- Stable Diffusion (sd-turbo) على كارت الشاشة — **0.9 ثانية للصورة** بعد أول تحميل
- اطلب بالعربي ("ارسم لي غروب على النيل") — بيترجم لوصف إنجليزي غني تلقائيًا
- الصورة **بتظهر جوة الدردشة** فورًا + معرض للصور المولدة

### 🖼️ فهم الصور (Vision)
- ارفع صورة أو سكرين شوت واسأل: "فيها إيه؟" / "اقرا النص اللي فيها"
- موديل رؤية مخصوص (`qwen2.5vl`) + OCR هجين للنصوص العربية والإنجليزية

### 📝 تقارير Word أكاديمية بمعايير APA 7
- صفحة عنوان، خط Times New Roman، تباعد مزدوج، هوامش 1 إنش، أرقام صفحات، مراجع مرتبة أبجديًا بـ hanging indent
- **التنسيق مفروض بالكود** مش متروك لمزاج الموديل

### ✍️ محتوى إبداعي
- قصص أطفال، قصص جريمة، كوميديا بالعامية المصرية، سيناريوهات أفلام كاملة بتقسيم المشاهد

### 🌐 بحث ويب ذكي — لما تحتاجه بس
- بيشتغل **بس** لما السؤال يحتاج معلومة حديثة (أخبار، أسعار، طقس) — مش catch-all
- أي سؤال معرفة عامة بيتجاوب محلي فورًا

### 🧠 ذاكرة على مستويين
- **قصيرة المدى**: بيفهم "طيب وهو؟" و"لخصهولي تاني" جوة نفس الجلسة
- **طويلة المدى**: قول "افتكر إن اسمي محمد" — هيفتكرها **حتى بعد إعادة التشغيل** (مجربة عبر عمليتين منفصلتين ✅)

### 🎭 شخصية حية مش روبوت
- **بيعكس أسلوبك**: رسمي ↔ رسمي، هزار ↔ هزار بإيموجيز 😄
- ردود السلام **إبداعية من الموديل نفسه** ومش بتتكرر: "صباح الكنافة بالمانجا! 😋" ثم "صباح الفل واللوز المحمص 🌼" — مع فلتر جودة بيمنع أي هلوسة
- مقدمات ذكية مع كل إجابة: "اتفضل يا فندم، الكود جاهز 📐✅"
- كارت تعريف فوري لـ "مين انت؟" وكارت المطور لـ "مين اللي عملك؟"

### 🕹️ تحكم كامل وقت التشغيل
- **زر إيقاف حقيقي** ⏹️ — وقّف الموديل في نص الشغل واكتب سؤال جديد فورًا
- **مؤشر حالة حي متحرك**: `🤔 بفكر ● ● ●` / `💻 بكتب الكود ● ●` / `🌐 ببحث في الويب ●` — عارف بيحصل إيه لحظة بلحظة
- **زراير اختيار المسار قبل الإرسال**: 🤖 تلقائي / 📄 من ملفاتي / 🌐 بحث ويب / 🧠 الموديل نفسه

### 🖥️ واجهة ويب أنيقة (Gradio)
- شات RTL كامل بالعربي، رفع وفهرسة ملفات بضغطة، تحميل نواتج الشغل (أكواد وتقارير)، معرض صور
- هيدر بـ badges، فقاعات دردشة ناعمة، زراير chips بتتفاعل مع الماوس

## 🛠️ التقنيات المستخدمة — وليه اخترناها

| الطبقة | التقنية | ليه بالذات؟ |
|---|---|---|
| 🤖 **إطار الوكلاء** | `smolagents` + **`ToolCallingAgent`** | خفيف ومرن. اخترنا `ToolCallingAgent` مش `CodeAgent` **بالتجربة**: موديل 7B بيفشل باستمرار في إنتاج بلوكات `<code>` سليمة مع CodeAgent |
| 🧠 **الموديلات (ثنائي)** | `qwen2.5:7b` عام + `qwen2.5-coder:7b` للبرمجة عبر **Ollama** | أفضل موديلات 7B مفتوحة بتدعم العربي. موديل مخصوص للكود بيدي جودة أعلى بكتير، و Ollama بيبدلهم في الـ VRAM تلقائيًا |
| 📄 **قراءة PDF** | `PyMuPDF` + **Tesseract OCR** fallback | أسرع مكتبة PDF، والـ OCR (عربي+إنجليزي) بيشتغل تلقائيًا لما الصفحة تطلع ممسوحة ضوئيًا (أقل من 20 حرف نص أصلي) |
| 🔍 **البحث الكثيف** | `multilingual-e5-small` | **قرار بالقياس**: الـ embeddings الإنجليزية القديمة (bge-small) كانت بتضيّع كل الأسئلة العربية — بعد التبديل: dense MRR قفز **0.650 → 0.875** |
| 📚 **البحث المتفرق** | BM25 (`rank_bm25`) + keyword fallback | بيمسك الكلمات المفتاحية الحرفية اللي الـ embeddings بتفوتها ("المراجع والمصادر") |
| ⚖️ **دمج النتايج** | Weighted RRF (أوزان 3.0/1.0/0.5) | **قرار بالقياس**: الدمج المتساوي كان بيسحب النتايج لتحت (0.792 < 0.875) — الأوزان المقاسة رفعت الـ pipeline لـ **MRR 0.967** |
| 🎯 **إعادة الترتيب** | `bge-reranker-base` (cross-encoder) | آخر تمريرة دقة على المرشحين + بيشغّل الـ routing الدلالي (هل السؤال عن المستندات أصلاً؟) |
| 🗄️ **قاعدة المتجهات** | ChromaDB (`PersistentClient`) | محلية، دائمة على الديسك، مجانية، صفر إعداد |
| 🎨 **توليد الصور** | Stable Diffusion `sd-turbo` عبر `diffusers` | نسخة distilled: 4 خطوات بس، fp16، بتدخل في 6GB VRAM — **0.9 ثانية للصورة** |
| 👁️ **فهم الصور** | `qwen2.5vl:7b` + OCR هجين | vision-language موديل محلي للوصف + Tesseract للنصوص الدقيقة |
| 📊 **تحليل الجداول** | `pandas` | الأسئلة الحسابية ("متوسط لكل حي") محتاجة **حساب فعلي** مش استرجاع نصي |
| 📝 **تقارير Word** | `python-docx` | تنسيق APA 7 مفروض ببرمجة صريحة — الموديل بيكتب المحتوى بس |
| 🖥️ **الواجهة** | Gradio 6 | RTL جاهز، chat interface بـ streaming و generator support (زر الإيقاف والحالة الحية) |
| 🧪 **التقييم** | benchmark مخصوص (20 سؤال ar/en) + LLM-as-judge | **مبدأ المشروع الأساسي: قيس قبل ما تغيّر** — اتبنى قبل أي تحسين |
| 🩹 **إصلاح runtime** | `patches.py` | موديلات الكود المحلية بتطلع triple-quoted strings جوة JSON باستمرار — الباتش بيصلحها فورًا بدل ما الوكيل يحرق كل خطواته |

## 🏗️ المعمارية

```
                         ┌─────────────────────────┐
                         │     🧠 Manager Agent     │
                         │  فاست-باث فوري + سلسلة   │
                         │   أولويات + ذاكرة + أمان  │
                         └────────────┬────────────┘
        ┌──────────┬──────────┬───────┼───────┬──────────┬──────────┐
        ▼          ▼          ▼       ▼       ▼          ▼          ▼
   ┌─────────┐┌─────────┐┌────────┐┌───────┐┌─────────┐┌────────┐┌────────┐
   │📄 RAG    ││👁️ Vision ││📝Report ││💻Coder ││🎨Creative││🌐 Web   ││🧠 Base  │
   │ Agent    ││ Agent    ││ Agent   ││ Agent  ││ Agent    ││ Agent   ││ Model   │
   │ بحث هجين ││ وصف+OCR  ││ APA 7   ││كود كامل││قصص+صور   ││أخبار    ││معرفة    │
   │ + rerank ││          ││ Word    ││+ فحص   ││+سيناريو  ││وأسعار   ││عامة     │
   └─────────┘└─────────┘└────────┘└───────┘└─────────┘└────────┘└────────┘
```

**سلسلة الـ Routing الذكية** (بالترتيب):
1. ⚡ **فاست-باث فوري (صفر LLM)**: كارت المطور → كارت الهوية → سلام (رد إبداعي بنداء مباشر واحد) → أسئلة الملفات metadata → أوامر "افتكر"
2. 🎯 **كشف نية صريح**: شرح كود ملزوق → فهم صور → مستندات (فحص دلالي بالـ reranker) → جداول → تقارير → برمجة → إبداع
3. 🛟 **أمان مضمون**: no-match بيكمل للخطوة اللي بعدها، أي خطأ بيرجع للموديل الأساسي — **المستخدم دايمًا بياخد إجابة**

## 🔬 قرارات هندسية اتاخدت بالقياس (مش بالإحساس)

1. **📏 التقييم اتبنى الأول** — قبل أي تحسين. كل تغيير في الاسترجاع بيتقاس قبله وبعده، والتاريخ كله محفوظ في `eval/RESULTS.md`
2. **🔄 الـ embeddings اتبدلت بالدليل** — الأرقام ورّت إن كل الأسئلة العربية بتضيع، فاتبدلت واتقاس الفرق (+0.225 MRR)
3. **⚖️ أوزان الدمج من sweep مقاس** — مش أرقام عشوائية: تجربة منهجية على الـ benchmark
4. **🚫 الدمج الدلالي للذاكرة اترفض بالقياس** — اتجرب واتقاس وطلع مش بيميّز على الأسئلة العربية القصيرة، فرجعنا للحقن المباشر
5. **⚡ فاست-باث بدل موديل** — الـ 7B بيرد نص عادي بدل tool call على المدخلات الصغيرة (كان بيحرق 20-30 ثانية على "اهلا") — الحل: ردود فورية بفلتر جودة
6. **🔒 كل مسار من الـ LLM بيتفحص** — أسماء ملفات بتتحقق ضد path traversal، أكواد بايثون بتتفحص AST قبل الحفظ، وسطور الموديل الإبداعية بتتفلتر ضد الهلوسة

النواة متلمستش — الـ backend مجرد wrapper رفيع (القرار الدائم #5):

- 🌐 **FastAPI backend** (`server/`): JWT (access + refresh) + تأكيد إيميل بـ **OTP عبر Gmail SMTP** + PostgreSQL (بتتعمل تلقائيًا أول تشغيل) لتخزين المستخدمين والمحادثات والرسايل
- 🔑 **حسابات كاملة**: تسجيل → OTP → خطوة "اسمك إيه؟" (الاسم بيتسجل في ذاكرة NilePi + اختيار أفتار فرعوني 🎭) + **نسيت كلمة السر** بكود إعادة تعيين (إيميله مختلف بصريًا عن إيميل التأكيد — كهرماني تحذيري) + قواعد صارمة front + back: باسورد 8+ حروف إنجليزية وأرقام ورموز بس، ويوزرنيم **إنجليزي وأرقام و `.` و `_` بس** (3-30 حرف) بتنبيه حي وانت بتكتب
- ✨ **صفحة دخول حية**: الـ 👋 بيلوّح فعلًا، والإيموجي بيتبدل 🙈/👀 مع خانة الباسورد + **"أهلاً محمد" لآخر يوزر دخل من الجهاز** (شريحة باسمه وأفتاره وزرار "مش انت؟" — الفوكس بيروح للباسورد على طول) + تبويبات دخول ↔ تسجيل بتنزلق أفقي (RTL-aware) + **أنيميشن نجاح قبل التحويل**: دايرة gradient بتنط spring مع حلقات موج نيلية 🌊 و"أهلاً بيك" + خانات الـ OTP بتفرح بالأخضر (قفزة متتابعة) لو الكود صح وبتحمرّ وتهتز وتتفضى لوحدها لو غلط + تحذير Caps Lock + تشيك ليست حية لشروط الباسورد + كونفيتي تفعيل الحساب 🎉
- 👥 **عزل كامل لكل مستخدم**: Chroma collections خاصة (`user_{id}_documents` / `user_{id}_memory`) + مجلدات `data/users/{id}/` منفصلة (uploads / workspace / صور) + قفل GPU بيسلسل الطلبات (كارت واحد = طلب واحد في اللحظة)
- ⚛️ **واجهة Next.js احترافية** (`web/`): landing page تقديمية بالعربي (سكشن معمارية تفاعلي بخطوط توصيل متحركة ووكلاء بألوان مميزة، عدادات متحركة للأرقام، فواصل موج متحركة بين السكاشن، مشهد خصوصية SVG تفاعلي، وكارت المطور **بيتفرد زي لفافة بردي** 📜 أول ما يدخل الشاشة) + صفحات تسجيل/دخول بكارت زجاجي وخلفية متحركة ومؤشر قوة الباسورد ونقاط تقدم + تطبيق شات كامل (سايدبار محادثات، **بحث في المحادثات 🔍** بالعناوين والرسايل مع مقتطف حوالين الكلمة، رفع بسحب وإفلات، **تصوير مباشر بكاميرا الموبايل 📸**، Markdown + syntax highlighting، الحالة الحية عبر SSE، زر إيقاف، معرض صور، تحميل النواتج) + **وضع فاتح/غامق** 🌓 + **إشعارات متصفح 🔔** لما الرد يجهز وانت في تاب تاني + مشاركة محادثة برابط قراءة فقط بكارت OG بعنوان المحادثة نفسها
- 🎧 **خدمة عملاء وإشعارات**: زرار عايم في الشات — المستخدم بيبعت تذكرة دعم (بتوصل نوتيفيكيشن للأدمن)، الأدمن بيغير حالتها (قيد المراجعة → جاري الحل → تم الحل / مغلقة) ويضيف ملاحظات بتوصل لصاحبها كإشعار، مع بادج بعدد غير المقروء (polling)
- 🛡️ **Admin panel** على `/admin` (صاحب `ADMIN_EMAIL`): إحصائيات + **رسوم بيانية SVG** لنمو المستخدمين والرسائل يوميًا (آخر 30 يوم) + **رسالة عامة 📢** بتوصل نوتيفيكيشن لكل المستخدمين + **شريط إعلان عام بأرشيف كامل 🗂️**: الأدمن بيشوف كل الإعلانات القديمة ويرجّع أي واحد للشريط أو يوقّف النشط أو يمسح نهائيًا (واحد بس نشط في نفس الوقت) — والمستخدم لما يقفل الشريط بيفضل مقفول طول الجلسة ويرجع يظهر لو الإعلان اتعدل + إدارة المستخدمين (تفعيل/تعطيل، تغيير باسورد، حذف نهائي بكل البيانات) + إدارة تذاكر الدعم
- 🧪 اختبارات كاملة: مسار الـ auth (register → OTP → verify → login → refresh → forgot/reset) + **قواعد اليوزرنيم والباسورد** + دورة التذاكر والإشعارات كاملة + الرسالة العامة + رسم النمو + المشاركة والتصدير + **البحث في المحادثات** (تطابق عنوان/رسالة، مقتطف، عزل مستخدمين) + البروفايل والصوت — بـ SMTP بـ mock

**التشغيل:** `python -m uvicorn server.main:app --port 8000` ثم `cd web && npm run dev` → http://localhost:3000
**الإعداد:** انسخ `.env.example` لـ `.env` واملأ `DATABASE_URL` و `JWT_SECRET` و `GMAIL_USER`/`GMAIL_APP_PASSWORD` و `ADMIN_EMAIL` — لينكات السوشيال في `web/lib/site.ts`
